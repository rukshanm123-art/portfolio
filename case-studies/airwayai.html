AirwayAI — Lung Cancer Detection Web Application

Final Year BSc Project — AI-powered deep learning system for automated lung nodule detection and classification from CT scans.

Project Overview

AirwayAI is an end-to-end AI system developed to assist healthcare professionals in detecting lung nodules from CT scan images. The platform integrates a Convolutional Neural Network (CNN) model with a secure web application interface for image upload, classification, and report generation.

Problem Statement

  • Manual interpretation of CT scans is time-consuming.
  • Diagnostic variability can lead to missed or delayed detection.
  • Radiologists face increasing workload pressure.

Solution

  • Built a CNN-based deep learning model for lung nodule detection.
  • Integrated model into a secure web application.
  • Implemented automated malignancy probability scoring.
  • Generated structured diagnostic reports.
  • Enabled user authentication and scan history storage.

Technology Stack

Python TensorFlow / Keras CNN Flask React / JS MongoDB / MySQL DICOM Processing

Model Architecture

  • Input: Preprocessed CT scan images
  • Multiple Convolutional layers with ReLU activation
  • Max Pooling layers for feature reduction
  • Fully Connected Dense layers
  • Sigmoid output layer for binary classification
  • Loss Function: Binary Cross Entropy
  • Optimizer: Adam

Model Performance

  • Accuracy: ~90%+
  • Precision & Recall optimized for early detection
  • Reduced analysis time compared to manual review

Challenges Faced

  • Preprocessing large DICOM datasets
  • Handling data imbalance
  • Preventing overfitting during model training
  • Integrating AI inference with real-time web UI

Impact & Significance

AirwayAI demonstrates the practical application of artificial intelligence in medical diagnostics. The project showcases full-stack integration of deep learning models with a secure web-based interface, highlighting skills in AI development, backend engineering, frontend design, and system architecture.
